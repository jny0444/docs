> *Esta página é fornecida para fins informativos para explicar a mecânica do protocolo. Não constitui uma oferta de valores mobiliários ou aconselhamento de investimento. $P2P não está disponível para pessoas dos EUA. Nada aqui deve ser interpretado como uma promessa de retorno financeiro. Por favor, leia a [Divulgação completa](/pt/for-token-holders/disclosures) antes de prosseguir.*

20% das taxas do protocolo são contribuídas para o tesouro, planejado para aumentar para 35% conforme o protocolo amadurece, sujeito à governança de futarquia MetaDAO.

- 20% das taxas do protocolo fluem para o tesouro on-chain (aumento planejado para 35% via governança de futarquia MetaDAO)
- Via governança de futarquia MetaDAO, os detentores de tokens decidem como os fundos do tesouro são implantados, incluindo desenvolvimento de ecossistema, operações de liquidez ou outras ações que servem o protocolo
- Toda implantação do tesouro requer aprovação de governança e está sujeita ao mecanismo de futarquia
- A primeira ativação de governança do tesouro está planejada para aproximadamente 3 meses após a TGE, sujeita à prontidão da governança

Os detentores de tokens governam como os fundos do tesouro são implantados. Todas as ações do tesouro requerem aprovação de governança através do mecanismo de futarquia. Nenhuma implantação de tesouro é automática ou garantida. Nenhum detentor individual de tokens tem uma reivindicação sobre os fundos do tesouro.
